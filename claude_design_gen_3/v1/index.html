About.

I’m a full-stack developer based in Gdańsk. For 1.5 years I worked on a SaaS in a three-person team where everyone shipped everything: backend in PHP, frontend in JavaScript and React, infra, copy, the lot.

Before that, three months building an indoor navigation app for a hospital. React Native, a 3D model of the building, Dijkstra for pathfinding. Real users, real corridors, real stakes.

I like the kind of work where you own a feature from database schema to the last hover state. I like products that respect the user’s time.
